Paramètres

  • title (type: string)
    Titre alternatif qui sera affiché dans le composant Summary.
  • anchor-value (type: string)
    Valeur qui sera affichée sous l'ancre.
  • anchor-error (type: boolean)
    Définit si la valeur est dans un état d'erreur.

Checklist

Exemple

Paramètres

  • ng-model (type: array)
    Tableau des éléments à afficher dans la liste.
  • min-length (type: number, default: 3)
    Nombre de caractères nécessaires pour que la recherche soit prise en compte.
  • max-tags (type: number, default: 5)
    Nombre maximum de tags à afficher avant que l'overflow ne s'active.
  • placeholder (type: string)
    Label s'affichant dans le champs de recherche quand il est vide.
  • empty-label (type: string, default: 'No result)
    Label s'affichant lorsque la liste des éléments est vide.
  • selection-label (type: string, default: 'Select elements containing:)
    Label précédant les boutons de sélection.
  • items-length-label (type: string, default: 'Number of items:)
    Label précédant le nombre d'éléments dans la liste.
  • select-all-label (type: string, default: 'Include)
    Label du bouton de sélection.
  • unselect-all-label (type: string, default: 'Exlude)
    Label du bouton de déselection.
